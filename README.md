# Astro Blaster 🚀

Juego arcade espacial hecho con HTML5 Canvas y Electron.

## Cómo jugar
- `←` `→` para mover la nave
- `ESPACIO` para disparar
- Destruye asteroides para ganar puntos
- El juego se acelera con el tiempo
- 5 kills seguidos = COMBO x5 + 100 pts bonus

## Tecnologías
- HTML5 Canvas
- JavaScript
- Electron

## Instalación local
npm install
npm start


## Build ejecutable
npm run build


## Jugar en el navegador
Abre `index.html` directamente en tu navegador.
